A simple full-stack app where workers list services and clients can book them.

Features :
Worker profile creation
Add skills, pricing, categories

Client registration/login  | Browse workers | Book services | JWT authentication | Tech Stack

Backend: Python, Django, DRF, JWT
Frontend: React, Tailwind, Axios

    cd backend

    cd ../frontend
    npm install
